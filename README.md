# 封神演义拓展

# 1.拓展背景
我在贴吧中没有找到任何有关于配置技能池的拓展以及相关封神演义的拓展，所以我想试着写一个既有技能池配置又有封神演义的拓展，拓展可以遵循着封神演义整部小说进行关卡模式和boss分配，一旦完成可能会是第一个类型的拓展不排除过程中有其他的拓展
